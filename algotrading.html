Algo Trading Bot

Created trading bot which place trade automatically on MT5 platform. Developed an API to place trade when certain event is trigger on Tradingview charts and it will call our api to place trades. Created some scripts for trading stretegies by using indecators like EMA and RSI. This stretegies will trigger alarm when condition to buy or sell trade will meet.
